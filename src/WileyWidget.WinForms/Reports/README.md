# Reports Templates

Drop FastReport `.frx` templates into this directory so the Reports view can load them at runtime.

The application currently ships with these templates:

- `BudgetComparison.frx`
- `BudgetForecastSummary.frx`
- `BudgetForecastLineItems.frx`

Any additional `.frx` file placed here will be copied to the build output and discovered by the Reports view at runtime.
