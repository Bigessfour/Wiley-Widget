# WileyWidget.WinUI Assets

This directory contains application assets for the WinUI 3 shell project.

## Required Assets

The following asset images are referenced in Package.appxmanifest:

- **Square150x150Logo.png** - Medium tile (150x150px)
- **Square44x44Logo.png** - App icon (44x44px)
- **Wide310x150Logo.png** - Wide tile (310x150px)
- **SplashScreen.png** - Splash screen (620x300px)
- **StoreLogo.png** - Store logo (50x50px)

## Placeholder Assets

Placeholder images will be generated during the first build. For production:

1. Replace placeholders with branded assets
2. Ensure PNG format with transparency
3. Follow Windows App SDK asset guidelines
4. Use high DPI assets for better scaling

## Asset Guidelines

- Use transparent backgrounds where appropriate
- Maintain consistent branding across all sizes
- Test on different display scales (100%, 125%, 150%, 200%)
- Follow Windows 11 design principles
