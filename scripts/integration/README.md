# Integration Scripts

Scripts for integrating with external services and APIs.

## QuickBooks Integration

- `setup-oauth.ps1` - Setup QuickBooks OAuth authentication
- `setup-quickbooks-sandbox.ps1` - Configure QuickBooks sandbox environment
- `setup-town-of-wiley.ps1` - Setup Town of Wiley QuickBooks configuration
- `test-qbo-secret-vault-integration.ps1` - Test QuickBooks secret vault integration
- `test-quickbooks-connection.ps1` - Test QuickBooks API connectivity
- `run-quickbooks-headless-test.ps1` - Run QuickBooks tests in headless mode
