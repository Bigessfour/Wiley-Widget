# Utility Scripts

General-purpose utility scripts for development and maintenance.

## Scripts

- `ps-runner.ps1` - PowerShell script runner with analysis
- `ps-safe-run.ps1` - Safe PowerShell script execution
- `psa-runner.ps1` - PowerShell Analyzer runner
- `Quick-Icon-Install.ps1` - Quick icon installation utility
- `test-vscode-config.ps1` - Test VS Code configuration
